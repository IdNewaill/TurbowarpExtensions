# [<Retour](../../../ReadMe.md) | Conversion de Charactère ASCII
Cette extensions permet de transformer des charactères et ASCII et inversement.

[➜ Télécharger extension](./ORD%20ET%20CHR.js)

## Blocs disponibles

### ![CHR bloc](./HideMe/newCHRbloc.svg)
    chr ( character : text ) : number

    ### Description
    Return the first character of the text ASCII number,
    But can return NaN (Scratch Error)
### ![ORD bloc](./HideMe/newORDbloc.svg)
    chr ( character : text ) : number

    ### Description
    Return the corresponding ASCII character,
    But can return NaN (Scratch Error)
